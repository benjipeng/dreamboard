Lucide Icons Examples

This page demonstrates different ways to use Lucide icons in a TypeScript project.

Method 1: Using data-lucide attribute

This is the simplest way to use Lucide icons. Just add a data-lucide attribute with the icon name.

<i data-lucide="home"></i>

home

settings

user

Method 2: Using data-lucide with custom attributes

You can customize the icon size, color, and stroke width.

<i data-lucide="alert-circle" data-lucide-size="32" data-lucide-color="red" data-lucide-stroke-width="1.5"></i>

alert-circle

check-circle

info

Method 3: Dynamically created with JavaScript

Icons can also be created dynamically using the createIcon utility.

const icon = createIcon('github', { size: 24 });
